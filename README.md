# Palico-Bot

Hello! This is a discord bot focused around Monster Hunter. (Only MHGU is supported for now)

# Commands
!key arg1 arg2
arg1 = type of quest (village, hub, or g)
arg2 = star

Example: !key hub 7

!item arg1 arg2
arg1 = item name
arg2 = location rank (hr, lr, or g)

Example: !item frog hr
